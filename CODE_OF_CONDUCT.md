# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and inclusive environment for everyone.

## Expected Behavior

- Be respectful
- Be collaborative
- Accept constructive feedback
- Support fellow contributors

## Unacceptable Behavior

- Harassment
- Discrimination
- Personal attacks
- Offensive language

## Enforcement

Project maintainers may remove comments, commits, or contributions that violate this code of conduct.
