# Contributing to AI Travel Planner

Thank you for your interest in contributing to AI Travel Planner!

## How to Contribute

1. Fork the repository.
2. Create a new feature branch.
3. Make your changes.
4. Test the application.
5. Commit your changes with meaningful commit messages.
6. Push your branch and create a Pull Request.

## Coding Standards

* Follow PEP 8 for Python code.
* Use descriptive variable and function names.
* Keep functions modular and reusable.
* Run pre-commit hooks before committing.

## Reporting Issues

If you find a bug or have a feature request:

* Create an issue with a clear description.
* Include screenshots or logs if applicable.
* Mention steps to reproduce the problem.

## Pull Request Guidelines

* Keep PRs focused on one feature or bug fix.
* Update documentation when necessary.
* Ensure all tests pass before submission.

Thank you for helping improve AI Travel Planner!
