# Project Constitution

## Purpose
AI Travel Planner helps users generate personalized travel itineraries using AI.

## Principles

1. Keep code simple and maintainable.
2. Store secrets in .env files.
3. Use pre-commit hooks before committing.
4. Follow spec-driven development.
5. Document new features under specs/.

## Project Structure

- pages/ for Streamlit pages
- utils/ for reusable functions
- specs/ for feature specifications
- outputs/ for generated files
